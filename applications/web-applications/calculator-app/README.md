# Calculator App

A fully functional calculator web application created using the Sky Force codex system.

## Features

- ✅ Basic arithmetic operations: addition, subtraction, multiplication, division
- ✅ Clear button to reset calculations
- ✅ Decimal number support
- ✅ Backspace functionality
- ✅ Error handling for division by zero
- ✅ Keyboard support for all operations
- ✅ Responsive design for mobile and desktop
- ✅ Modern, intuitive UI with smooth animations

## Files

- `index.html` - Main HTML structure
- `style.css` - Modern styling with gradient background and smooth transitions
- `script.js` - Complete calculator logic with error handling

## Usage

1. Open `index.html` in a web browser
2. Use the on-screen buttons or keyboard:
   - Numbers: 0-9
   - Operations: +, -, *, /
   - Decimal: . or period key
   - Equals: Enter or =
   - Clear: Escape, C, or c
   - Backspace: Backspace key

## Technical Details

- **Frontend**: Pure HTML, CSS, and JavaScript (no frameworks required)
- **Design**: Modern dark theme with gradient background
- **Responsive**: Works on mobile and desktop devices
- **Error Handling**: Graceful handling of division by zero and large numbers
- **Precision**: Floating-point arithmetic with proper rounding
