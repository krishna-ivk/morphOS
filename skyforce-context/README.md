# [ADJACENT] Skyforce Context

`skyforce-context` is a lightweight workspace context area, not a primary runtime repo.

## Current Role

Use this directory for:

- agent-specific context notes
- scratchpads
- lightweight workspace memory that supports active sessions

Do not treat this directory as:

- the canonical long-term memory platform
- the primary context API
- a replacement for shared contracts in `morphOS` or `skyforce-core`

## Current Contents

- `agents/`
  - lightweight agent notes and local context
- `scratchpads/`
  - active work-in-progress notes before they become formal artifacts

## Status

This directory is adjacent support material for the current workspace.
It is useful, but it is not part of the primary Skyforce product spine.
