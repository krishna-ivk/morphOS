# Pi Governed Builder Skill Pack

This package provides the transitional skill that keeps Pi aligned while Skyforce integrates Pi as a governed coding worker.

It is designed to attach primarily to the `coder` archetype and to reinforce the following posture:

- work from normalized factory contracts
- execute within bounded repo/workspace scope
- emit receipts and evidence
- propose follow-up work only as non-binding candidate contracts
