# morphOS Human Guide

`morphOS` is the specification repo for the long-term operating model of the platform.

## What This Repo Is For

Use this repo when you need to understand:

- what the agentic operating system is supposed to become
- how workflows should be described
- how policy should work
- how agents are framed conceptually
- how runtime topology and evolution are supposed to behave

## What Lives Here

- `agents/`
  - human-readable agent archetypes
- `docs/`
  - the system specifications
- `vision/`
  - product vision
- `workflows/`
  - workflow templates used by Symphony

## Human Mental Model

`morphOS` should answer:

- what are we trying to build?
- what are the rules of the system?
- what are the safe boundaries?

It should not become a second runtime or a duplicate of Skyforce.
