# Demo Packets

This folder is for ad hoc operator demo packets and local proving-run artifacts
that are useful for workshops, walkthroughs, or temporary local testing.

It is intentionally separate from `docs/examples/`.

## Rule

- `docs/examples/` should contain only canonical reference packets
- `docs/demo/` may contain ad hoc or operator-created packets used for local demos

## Current Guidance

If a packet is referenced by a spec, runbook, or implementation checklist, it
belongs in `docs/examples/`.

If a packet was created for a one-off local run, manual showcase, or temporary
operator exercise, it belongs here.
